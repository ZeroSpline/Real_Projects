Важно выбрать фильм, который подходит студенту по уровню сложности, т.ч. студент понимал 50 - 70 % диалогов. Цель проекта состоит в разработке ML решения для автоматического определения уровня сложности англоязычных фильмов, а также подготовки интерфеса на базе streamlit для презентации заказчику.|
